# これは何？
- ed25519の公開鍵認証でパスワードなしでssh接続を実現する。
- gitを安全に効率よく操作することができるようになります。
- 例えば、、、
    - gitからcloneするたびにid,passを入力しなくてよくなる
    - gitにpushするたびにid,passを入力しなくてよくなる
